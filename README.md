# 🎨 Paisaje en Canvas 2D – Árboles, Río y Animales

## 📖 Descripción General

Este proyecto utiliza el elemento **`<canvas>` de HTML5** junto con **JavaScript** para dibujar un **paisaje completo** con:

* Cielo con **degradado azul**
* Montañas con **profundidad y capas**
* Suelo y pasto con **variaciones de color**
* Árboles **tipo redondo y pino**
* Río con **curvas y ondas**
* Puente, flores, piedras y animales (conejo y ardilla)

El objetivo es demostrar cómo combinar **formas geométricas básicas**, degradados, sombras y curvas Bézier para crear un **escenario visual detallado y armonioso** usando programación gráfica.

---

## ⚙️ Funcionalidades

* Renderiza automáticamente un **paisaje completo** al abrir la página.
* Dibuja **montañas, suelo, río y puente** con polígonos y curvas Bézier.
* Árboles híbridos:

  * Tipo redondo (círculos superpuestos)
  * Tipo pino (triángulos escalonados)
* Agrega **animales** (conejo y ardilla) con detalles de cuerpo, cabeza, orejas y ojos.
* Dibuja **flores y piedras** para enriquecer el entorno.
* Aplica **degradados y sombras** para sensación de profundidad y realismo.
* Todo el estilo visual está definido en **JavaScript**, manteniendo la separación entre diseño y lógica.

## 📸 Vista Previa (Descripción)

Al abrir el proyecto en un navegador, se visualiza un **paisaje natural completo**:

* Cielo azul con degradado suave
* Montañas con diferentes capas de color
* Pasto y suelo con tonos variados
* Árboles de distintos tipos distribuidos por el terreno
* Río con curvas y ondas que atraviesa el paisaje
* Animales, flores y piedras distribuidos para dar realismo
